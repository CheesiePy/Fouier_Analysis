                let W be a vector space over a field F
                Inner product of two vectors u,v in W
                (*,*) ; WxW -> F
                such that:
                    1. < u,v > >= 0
                    2. < u,v > = 0 iff u = 0 or v = 0
                    3. complex conjugate - < u,v> = < u,v>
                
                for all a,b in the complex numbers
                and all u,v,w in W
                < a*u + b*v, w> = a*< u,w> + b*< v,w>

                

                (*,*) is a bilinear map
                (*,*) is symmetric
                (*,*) is positive definite

                example:
                a. W = R^2
                x = (x1,x2) and y = (y1,y2)
                < x,y> = x1*y1 + x2*y2
                this is a dot product
            
                proff:
                1. < x, x> = x1^2 + x2^2 >= 0

                1. < x, x> = 0 iff x = 0

                2. < a*x + b*y, z> = <(a*x1 + b*y1, a*x2 + b*y2), (z1, z2)>
                = a*x1*z1 + b*y1*z1 + a*x2*z2 + b*y2*z2
                = a(x1*z1 + x2*z2) + b(y1*z1 + y2*z2)
                = a*< x,z> + b*< y,z>

                3. < x, y> = x1*y1 + x2*y2 = y1*x1 + y2*x2 = < y,x>
                note: complax conjugate is the same number in the real number field

                example:
                b. W = C^3
                x = (x1,x2,x3) and y = (y1,y2,y3)
                y~ = conjugate of y
                < x,y> = sigma = for(i=1 to 3) {x_i*(~y_i)}
                prove this is a inner product:
                1. < x, x> = sigma() = for(i=1 to 3) {x_i*(~x_i)} = |x1|^2 + |x2|^2 + |x3|^2 = sigma(|xi|^2) >= 0

                2. let x,y,z be vectors in C^3 and a,b be complex numbers
                < a*x + b*y, z> = sigma() = for(i=1 to 3) {a*x_i*(~z_i) + b*y_i*(~z_i)}

                3. < x, y> = sigma() = for(i=1 to 3) {x_i*(~y_i)} = sigma(y_i*(~x_i)) = < y,x>

                reminder:
                    the conjugate of a multiplication is the same as the multiplication of the conjugates
                reminder:
                    the conjugate of a sum is the same as the sum of the conjugates
